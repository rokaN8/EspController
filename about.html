Inspiration & Vision

üßó Moonboard Training System

This project serves as a proof-of-concept for creating a home bouldering training system inspired by the Moonboard - a standardized climbing training board used worldwide. The Moonboard features a grid of climbing holds with LED indicators that light up to show specific boulder problems (routes), allowing climbers to train consistently using the same problems regardless of location.

üè† Home Training Vision

The goal is to develop an affordable, DIY version of commercial LED climbing systems for home use. By using readily available components like ESP32 microcontrollers and addressable LED strips, climbers can create their own interactive training boards.

üìà Scalable Design

While this 5√ó6 grid demonstrates the core technology with 30 LEDs, the system can be scaled to match standard training board dimensions:

  • Custom Walls: Any configuration for home setups

The modular approach allows climbers to start with a small prototype and expand their setup over time as they add more holds to their home wall.

Technical Specifications

Hardware Requirements

  • Microcontroller: ESP32 development board (any variant)
  • LEDs: WS2811 addressable LED strip (30 LEDs)
  • Power: 5V external power supply (~1.8A for full brightness)
  • Connections: Data pin (GPIO 4), Power, Ground

Software Stack

  • Frontend: HTML5, CSS3, JavaScript (Web Bluetooth API)
  • Backend: Arduino C++ with ESP32 libraries
  • Communication: BLE (Bluetooth Low Energy)
  • LED Control: FastLED library
  • Data Format: JSON command protocol

Browser Support

  • Full Support: Chrome, Edge (desktop and mobile)
  • Limited Support: Firefox (requires manual flags)
  • Not Supported: Safari

Usage Examples

Basic LED Control

Click any square in the LED grid to cycle through colors: Off ‚Üí Red ‚Üí Green ‚Üí Blue. Click "Send Configuration" to update the physical LEDs.

Test Mode

Use "Test LEDs" to sequentially light up each LED in red, helping you verify connections and identify any faulty LEDs.

Disco Mode

Activate "Disco Mode" for an automated light show that randomly selects 5 LEDs and assigns random colors every 2 seconds.

Brightness Adjustment

Use the brightness slider to adjust LED intensity from 5% to 100%, perfect for different ambient lighting conditions.
